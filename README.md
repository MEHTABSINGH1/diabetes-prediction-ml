# diabetes-prediction-ml
# Diabetes Prediction using Machine Learning

This project predicts whether a person is diabetic or not using the Pima Indians Diabetes Dataset.  
It is a classification problem solved using Machine Learning in Python.

## Problem Statement
Diabetes is a common health condition. Early prediction can help in better treatment.  
This project builds a machine learning model that predicts diabetes based on medical attributes.

## Dataset
Pima Indians Diabetes Dataset

Features used:
- Pregnancies
- Glucose level
- Blood Pressure
- Skin Thickness
- Insulin level
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age

Target:
- 0 → Non-diabetic
- 1 → Diabetic
